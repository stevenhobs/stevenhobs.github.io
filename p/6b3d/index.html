Node

  • 缓存路径自己自定义

  • 强烈不推荐 使用npm，他只是个 yarn下载器

nvm-windows

安装这个程序实现多node版本切换

  • 配置node的实体位置：nvm root [path]

  • 配置node的下载镜像站：nvm node_mirror https://npm.taobao.org/mirrors/node/

  • 配置npm的下载镜像站：nvm npm_mirror https://npm.taobao.org/mirrors/npm/ (虽然 .npmrc 设置过了)

切换命令一定要使用管理员权限执行

nvm

Unix 系统下的多node版本切换

Yarn

  • 在用户目录下新建 .yarnrc 文件

  • npm i -g yarn

  • yarn config set 变量名 变量值

    可配置属性

    变量名 变量值 说明
    regitstry “https://registry.npm.taobao.org” 仓库镜像源
    cache-folder “D:\\Cache\\yarn” 依赖缓存
    global-folder “D:\\Sdk\\yarn” 全局模块
    prefix “D:\\Sdk\\yarn” 模块可执行二进制文件

    小彩蛋，在 .yarnrc 添加一行 “--emoji true 执行yarn命令会有可爱的表情哦。

  • yarn 自举

    yarn global add yarn

    npm rm -g yarn

MySQL

官网下载 win-zip版 解压到指定目录，添加文件 my.ini:

  • 初始化data，管理员 mysqld --console --initialize
  • 注册服务，管理员mysqld --install
  • 删除服务，管理员mysqld --remove
  • 首次登陆，一定要重新设置root密码。
  • 推荐使用 第三方的python版 mysql 命令行客户端：mycli pip install mycli

MongoDB

新建文件 mongodb.conf

命令行启用数据库，必带配置参数 mongod --config ./mongodb.conf
