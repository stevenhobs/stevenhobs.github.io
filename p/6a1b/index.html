Something you need know before learning Web courses

浏览器

  • 浏览器始于网景公司的导航家（Navigator），现在生活离不开网页，网景已成历史，我们应铭记这个伟大的公司。

  • 现代主流的浏览器就三家，Chrome（谷歌）、firefox（Mozilla）、Safari（苹果），也具备动态更新的能力，我们可无缝体验新的标准和新的特性。

  • 浏览器的主要构成，文档排版引擎（通俗叫法 内核），JS引擎，CSS渲染引擎

    浏览器 内核 JS引擎
    Chrome Blink V8
    Mozilla Gecko SpiderMonkey
    Safari Webkit Webkit
    Edge（旧版） EdgeHTML Chakra Core
    IE9+ Trident / MSHTML Chakra

    微软也曾入过浏览器大战，惜败，新版Edge已使用Chromium（Chrome原型），IE和Edge较闻名，故附说。

W3C

万维网联盟，负责统一和维护浏览器的Web标准。Web标准是一种开发者与浏览器服务商的约定俗成，应该遵守大家都认可的，实际上各家都会带点私货。

从页面构成角度说，网页由 结构、表现、行为 三部分组成。

  • 结构：当前标准为 HTMl5，小版本修订 为 5.2（2017颁布）
  • 表现：CSS规范，最新为CSS3
  • 行为：DOM + ECMAScript

HTTP

网页通信使用的网络协议为HTTP一类，主要有

  • HTTP1.2
  • HTTP2

HTTP是基于TCP/IP通信协议（稳定可靠的链接），但是明文传输，后来有了加密认证，即 HTTPS

DNS

域名解析服务器，我们所有的请求地址最终都是 IP地址，故需要将 通俗意义的 请求链接 解析出 IP地址，DNS负责记录对映。

URI

统一资源标识符：用于标识某一互联网资源名称的字符串，最常见的形式是URL（统一资源定位符），罕见有URN（统一资源名）

文法

[协议名]://[用户名]:[主机名]@[主机名]:[端口]/[路径]?[查询参数]#[片段ID]
