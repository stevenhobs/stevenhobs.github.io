Windows Terminal使用
2021-10-29|开发
字数总计: 481|阅读时长: 1 分钟

Windows Terminal 是 windows 新开发的终端交互程序，可支持各种shell运行，支持GPU加速以及UTF-8，在Windows做开发，用之真香。

安装

Windows Terminal 可以在 Microsoft Store 中下载，稳定版即可

配置

官方的配置文件是 json 格式的，

主要可以配置的是终端全局设置，shell默认，独立shell的自定义设置，主题样式库和快捷键绑定

全局设置

属性名 属性值 用途
initialCols Int 窗口默认宽
initialRows Int 窗口默认高
initialPosition Str>Num 启动位置
centerOnLaunch Bool 居中启动，和上一条互斥
copyOnSelect Bool 选中即复制
copyFormatting Bool，”all”,”none”,”html”,”rtf” 复制附加格式信息
trimBlockSelection Bool 去除首尾空格
minimizeToNotificationArea Bool 最小化通知栏
alwaysShowNotificationIcon Bool 保留通知栏
multiLinePasteWarning Bool 多行粘贴警告
useAcrylicInTabRow Bool 标题栏亚克力背景
confirmCloseAllTabs Bool 关闭窗口提醒

默认配置 & Shell独立配置

属性名 属性值 说明
name Str Shell名
commandline Str 可执行命令
startingDirectory Str 初始路径
tabTitle Str Shell标签名
hidden Bool 选择列表隐藏
colorScheme Str 颜色样式
font Obj 字体系列设置对象
font.face Str 字体名
font.size Num 字体大小
font.weight Str 字重
font.features Obj 字体特性对象
font.axes Obj 字体格式对象
font.axes.ital Num 斜体
intenseTextStyle “none”,”bold”,”bright”,”all” 强调字符格式
cursorShape “bar”,”vintage”,”underscore”,”filledBox”,”emptyBox”,”doubleUnderscore” 光标形状
backgroundImage Str 背景图
backgroundImageOpacity Num[0,1] 背景图不透明度
icon Str 标签logo
useAcrylic False 背景亚克力材质
opacity Num[0,1] 不透明度
padding Num 内边距
scrollbarState “visible”,”hidden” 滚条状态
background Str 背景色
guid Str:guid 独立标识
bellStyle “all”,”audible”,”visual”,”none” 响铃

快捷操作

  • Ctrl + Shift + f 快捷搜索
  • Alt + Shift + plus + Alt + Shift + - 分拆窗格
  • Alt + Shift + D 分拆同源窗格（不可改方向）
  • Alt + UDLR 切换聚焦窗格
  • Alt + Shift + UDLR 调整窗格宽高
  • Ctrl + Shift + w 关闭聚焦窗格
